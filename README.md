# MiniPrayerTracker

MiniPrayerTracker is a simple Android application to help users track their daily Islamic prayers (Salat). It provides a simple interface to mark which of the five daily prayers have been completed.

## Features

*   Log which of the five daily prayers have been performed (Fajr, Dhuhr, Asr, Maghrib, Isha).
*   Display a summary of the day's completed prayers.
*   Calculate a completion percentage score based on the prayers performed.

## How to Use

1.  Launch the app.
2.  On the main screen, select the prayers you have completed for the day.
3.  Click the button to see the summary.
4.  On the summary screen, you can view your completed prayers and calculate your score.

## Project Setup

This is a standard Android Studio project. You can clone this repository and open it in Android Studio to build and run the app.
